# REST API
Ver. 1.1 </br>
Aplikacja umożliwiająca podstawowe operacje CRUD
## Info:
- Spring Boot 3.1.5
- Java 17
- Maven
### Dependencje:
- Spring Web
- Spring Data JPA
- H2 Database
#### Dodatkowe informacje
- Baza wypełniana jest przy użyciu @EventListener
- Podstawowe operacje GET, POST, PUT, DELETE
- Metody zwracają ResponseEntity
